# FEC-Project-Atelier

## Setup instructions:
  1. `npm install`

  2. Make a copy of `example.env` and rename to `.env`

  3. Create a .gitignore file and place in:
    - node_modules, client/dist, .env

  4. `npm run server-dev`

  5. `npm run client-dev`
    - listens on PORT set in `.env` or 3000

## Testing:
  1. `npm test`
